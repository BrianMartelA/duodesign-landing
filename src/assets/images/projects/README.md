# Imágenes de Proyectos

Esta carpeta contiene las capturas de pantalla y mockups de los proyectos completados.

## Archivos recomendados:
- `proyecto-corporativo.jpg` - Plataforma Corporativa
- `app-inmobiliaria.jpg` - App de Gestión Inmobiliaria

## Especificaciones:
- Formato: JPG o PNG
- Proporción: 16:9 (aspecto video)
- Tamaño recomendado: 1200x675px
- Peso máximo: 800KB
- Calidad: Alta resolución para efectos hover

## Uso en el código:
```typescript
// En about.component.html
src="assets/images/projects/proyecto-corporativo.jpg"
src="assets/images/projects/app-inmobiliaria.jpg"
```
