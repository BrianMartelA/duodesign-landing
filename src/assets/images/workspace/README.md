# Imágenes del Workspace

Esta carpeta contiene fotos del espacio de trabajo de los hermanos Martel.

## Archivos recomendados:
- `workspace-duo.jpg` - Foto principal del workspace

## Especificaciones:
- Formato: JPG o PNG
- Proporción: 1:1 (cuadrado)
- Tamaño recomendado: 800x800px
- Peso máximo: 600KB
- Calidad: Alta resolución para el efecto de rotación

## Uso en el código:
```typescript
// En about.component.html
src="assets/images/workspace/workspace-duo.jpg"
```
